Usability Assessment

Heuristics: How: Done:
I:Visibility of system status User will face an undefined session if they don't fill in all information ✔
II: Match between system and the real world Using maps and a pirate crest as background and logo to give it a sense of adventure and exploration ✔
III:User control and freedom User can skip questions and refuse to use location and camera ✔
IV: Consistency and standards Style design such as color pictures and background is consistent in across the app ✔
V: Recognition rather than recall Using location as a requirement to answer some questions ✔
VI: Flexibility and efficiency of use Use of cookies makes the app remember the session ✔
VII: Aesthetic and minimalist design Pictures are reused for every page of the app and most links are shown as images such as the facebook link and QR scanner link ✔
VIII: Help users recognize, diagnose, and recover from errors If users answer something wrong the game will display incorrect and if they press submit they will return to the question ✔
IX: Help and documentation Treasure Hunt facebook available as a link from the app to report errors ✔
